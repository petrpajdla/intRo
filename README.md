# R Intro Workshop

Materials and code for R workshop given at:

- CAA Vienna 2026

## Where to start?

Directory `code/` contains two scripts:

- `code_workshop.R` is the code with tasks and parts missing used at the workshop
- `code_full.R` is the full version for your reference containing *solutions*

Start with the `code_workshop.R` script in an empty project/directory.
Download the file and save it to `code` subfolder.
Data is downloaded in the script and directory structure is created on the fly.
